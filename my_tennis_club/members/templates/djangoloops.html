Excercises with "ifs" in Django

"if" excercise

{% for x in mymembers %} {% if x.firstname != "" %}
  • {{ x.id }} {{ x.firstname }} {{ x.lastname }}
    • {% else %}
  • No member
    • {% endif %} {% endfor %}

    Another "if" excercise

    {% if "Emil" in mymembersfnlist %}
  • Emil here
    • {% else %}
  • No such member
    • {% endif %} {% if "Roger" not in mymembersfnlist %}
  • Roger isn't here
    • {% else %}
  • Roger is here! Good boy!
    • {% endif %}

    "is" excercise

    What happens if we use "is" on objects with the same values

    {% if x is y %}

    YES

    {% else %}

    NO

    {% endif %}

    but what will happen if we add two variables which directs at the same object

    {% with var1=x var2=x %} {% if var1 is var2 %}

    YES

    {% else %}

    NO

    {% endif %} {% endwith %}

    What happens if we use "==" on objects with the same values

    {% if x == y %}

    YES

    {% else %}

    NO

    {% endif %}

    Excercises with loops in Django

    Usual for loop

    {% for x in mymembers %}
  • {{ x.id }} {{ x.firstname }} {{ x.lastname }}
    • {% empty %}
  • No members
    • {% endfor %}

    or we can show the same with another filling

    {% for x in mymembers %}
  • {{ x }}
    • {% endfor %}

    Usual reversed for loop

    {% for x in mymembers reversed %}
  • {{ x.id }} {{ x.firstname }} {{ x.lastname }}
    • {% empty %}
  • No member
    • {% endfor %}

    Empty excercise

    {% for x in mymember %} {% empty %}

    • {% endfor %}
